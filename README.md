# python_sample
Pythonのサンプルコード

# コメント

pythonではシャープ `#` で始める行がコメントとなる

# 型

## 数値

-  整数や小数で記述。例: 0, 1, -5, 1.5

### 数値メソッド

- as_integer_ratio: 小数を整数の比で表現

## 文字列

-  シングルクォートかダブルクォートで挟んだ文字列で記述。どちらでも意味は同じ。 例: "abc", 'def', ""

### 文字列メソッド

- upper: 大文字化
- lower: 小文字化
- find: 文字列中の部分文字列を検索
- isdecimal: 文字列が数値かどうか判定

# ブール型

- TrueかFalseのどちらかで表現。

# 演算子

## 算術演算
以下の演算子に対応

- 四則演算 (`+`, `-`, `*`, `/`)
- 剰余 (`%`)
- べき乗 (`**`)
- 整数除算 (`//`)

## 文字列操作

- 結合(`+`)
  - 数値の加算と同じ識別子だが、型によって判別される。
  - 文字列と数値を+で足そうとすると、エラーとなる


## 比較

- 大小比較 (`<`, `<=`, `>=`, `>`)
- 等値比較 (`==`)
- 不等比較 (`!=`)
## 代入

`変数名 = 値` で代入
変数名は英数字とアンダーバー(_)の組み合わせが使える。ただし数字で始めてはならない。

# 組み込み関数

## 算術

- abs: 絶対値
- round: 丸める
- min: 最小値
- max: 最大値

## 入出力

- print: データを表示。複数引数可能。
- input: データをコンソールから入力。引数は入力プロンプトに出すメッセージ。戻り値は入力内容。

## キャスト

- int: 整数化
- float: 実数化
- str: 文字列化

# import文

- モジュールを読み込む文
  - `import モジュール名` の形式
    - `モジュール名.関数名` で呼び出せるようになる


# 条件分岐

## if文

- `if 条件:` で条件分岐が可能
  - ブロックはインデントで表現。
  - ブロックは条件がTrueのときに実行
- `else:` で条件がFalseのときの処理も記述できる
- `elif 条件:` で複数条件の分岐が可能。

## よく使う関数

### 配列生成関数
- arange: 範囲とステップを指定してndArray配列生成
- array: 引数に配列やタプルを指定してndArray配列生成
- zeros: 形状を指定して値が0のndArray配列生成
- ones: 形状を指定して値が1のndArray配列生成
- empty: 形状を指定して値が未初期化(不定)のndArray配列生成

### ndArray型のメソッドなど

- dtype: 要素の型を取得
- ndim: 次元を取得
- size: サイズ(要素数)を取得
- itemsize: 要素1個のバイト長を取得
- shape: 配列の形状(次元とサイズ)を取得
- reshape: 配列の形状を組み替える
